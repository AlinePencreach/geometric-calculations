# geometric-calculations

Practicing JavaScript

## An application capable of calculating volumes, areas or perimeter/circumference of different geometric shapes (rectangle, circle, triangle, cube and spherical).

### choose the shapes on which you want to work:
- shows the different formulas available for this shape (calculation of air and volume for example).
- we can complete the formulas with our values
- displays the result.
